🚀 配置 Supabase 开启真实数据追踪（免费，5 分钟）

当前使用 浏览器本地存储——只能看到你自己浏览器的访问记录，不是真实流量。 配置 Supabase 后可以追踪所有用户的真实访问数据：

  1. 打开 supabase.com → 注册账号 → 创建免费项目
  2. 进入 SQL Editor → 新建查询 → 粘贴下方 SQL → 点击 Run
  3. 进入 Settings → API → 复制 Project URL 和 anon public key
  4. 打开 js/tracker.js → 替换 YOUR_PROJECT_ID 和 YOUR_SUPABASE_ANON_KEY
  5. 将 USE_SUPABASE 改为 true
  6. 同时更新本文件的 SUPABASE_URL 和 SUPABASE_ANON_KEY，将 SUPABASE_ENABLED 改为 true
  7. 刷新本页面 → 顶部角标变为绿色，即表示已连接真实数据
-- 在 Supabase SQL Editor 中执行以下 SQL：
CREATE TABLE pageviews (
  id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
  page TEXT NOT NULL,
  referrer TEXT DEFAULT 'direct',
  screen_w INT,
  screen_h INT,
  lang TEXT,
  timestamp TIMESTAMPTZ DEFAULT NOW()
);

-- 启用行级安全
ALTER TABLE pageviews ENABLE ROW LEVEL SECURITY;

-- 允许任何人写入（供追踪脚本使用）
CREATE POLICY "allow_insert" ON pageviews
  FOR INSERT WITH CHECK (true);

-- 允许任何人读取（供管理后台使用）
CREATE POLICY "allow_select" ON pageviews
  FOR SELECT USING (true);
